What is source control?

A component of software configuration management, version control, also known as revision control or source control, is the management of changes to documents, computer programs, large web sites, and other collections of information.

Why do I care?

  • Allows you to ensure that your code is in a working state
  • Prevents your computer from being the be all end all for housing your in-progress code
  • Make collaboration easier and attempts to prevent merge conflicts

What is Git?

What is Github?

Git and Github are NOT the same!

Git is the version control system.

Github is the user interface that utilizes git. You can use git without Github.

Vocabulary

  • Local: The machine you're working on
  • Remote: The server where the code lives that is not your computer
  • Repository: Folder/Directory
  • Commit: Defined version of your code and the changes you are making
  • Sync
    • Pushing: Moving code from your machine to the server
    • Pulling: Bringing changes down to your machine from the server
